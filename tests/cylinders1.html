Cylinders

Cylinders support up to 18 faces, although one face is reserved for the top and one for the bottom, so in practice they have a maximum of 16 side planes.

This limit is practical rather than theoretical: extra faces grow the size of the CSS file quite quickly. The maximum supported faces can be increased in the Sass file by changing the $max_faces variable.

This is an extremely short cylinder that resembles a disc more than a cylinder. It is set to be rolling rather than spinning because as a cylinder, it's not naturally face-on and so you'd only see the thin edge.

I think there's actually no current way to get a 3D disc-like shape (other than a 2-face object with border-radius) that spins. It's probably worth creating a "disc" which is to a cylinder what a wheel is to a carousel.
