Reflective Spaces

If a space has the class "reflective", then a mirror image of the objects inside the space is produced below it, giving the impression of a reflective floor that the objects are hovering over.

The gradient that fades out the reflection uses an SVG mask rather than a CSS gradient. A CSS gradient would need to start at transparent and fade to specific colour (e.g. white if the background is white). Because it is a mask, it works on any background colour.
